To get this project into your build, add following to your pom.

```xml
<repository>
  <id>jitpack.io</id>
  <url>https://jitpack.io</url>
</repository>
```
```xml
<dependency>
  <groupId>com.github.matsim-org</groupId>
  <artifactId>Opdyts-MATSim-Integration</artifactId>
  <version>-SNAPSHOT</version>
</dependency>
```
